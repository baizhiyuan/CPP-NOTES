# Contributor Covenant Code of Conduct

## 我们的承诺

为了营造一个开放友好的环境，我们作为贡献者和维护者，承诺让每个人参与本项目和我们社区的体验都不被骚扰，无论年龄、体型、可见或不可见的残障、族裔、性别特征、性别认同与表达、经验水平、教育背景、社会经济地位、国籍、个人形象、种族、宗教或性认同与取向。

我们承诺以促进开放、欢迎、多元、包容、健康的社区方式行事并互动。

## 我们的标准

有助于积极环境的行为示例包括：

- 对他人表现同理心和善意
- 尊重不同的意见、观点和经历
- 给出和优雅地接受建设性反馈
- 为我们的错误负责并向受影响的人道歉，从中吸取经验
- 关注的不仅是个人，更是整个社区的整体利益

不可接受的行为示例包括：

- 使用带有性意味的语言或图像，以及任何形式的性关注或挑逗
- 钓鱼、人身攻击或政治攻击、辱骂或贬损性评论
- 公开或私下骚扰
- 未经明确许可发布他人的私人信息（例如住址、邮箱）
- 其他在专业环境中合理认定为不当的行为

## 执行责任

社区维护者负责澄清和执行可接受行为标准，对任何被认为不当、威胁、冒犯或有害的行为采取适当且公平的纠正措施。

社区维护者有权删除、编辑或拒绝与本行为准则不符的评论、提交、代码、wiki 编辑、议题和其他贡献，并在适当时通报缘由。

## 适用范围

本行为准则适用于所有社区空间，亦适用于个人在公共空间正式代表社区的场合。

## 执行

辱骂、骚扰或其他不可接受行为可向项目所有者 [@baizhiyuan](https://github.com/baizhiyuan) 通过 GitHub Issue 报告。所有投诉将得到及时和公正的审查与调查。

## 执行准则

社区维护者将依据以下《社区影响指南》决定违规行为的后果：

### 1. 更正
**影响**：使用了不当的语言或行为。
**后果**：维护者私下书面警告并澄清。

### 2. 警告
**影响**：通过单次事件或一系列行为违规。
**后果**：附后果的警告。

### 3. 临时封禁
**影响**：严重违反社区准则。
**后果**：一段时间内禁止与社区互动。

### 4. 永久封禁
**影响**：表现出系列违规模式。
**后果**：永久禁止公开互动。

## 出处

本行为准则改编自 [Contributor Covenant 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct.html)。社区影响指南受 [Mozilla 行为准则执行阶梯](https://github.com/mozilla/diversity) 启发。
